# Nivalis Plotter

## Installation
- Install nana <https://github.com/qPCR4vir/nana>
- Compile with CMake: `cd build && cmake .. -DCMAKE_BUILD_TYPE=Release'

## Usage 
- `./nivalis`
- Enter expressions to evaluate then
- Enter `plot' to show plotter GUI
